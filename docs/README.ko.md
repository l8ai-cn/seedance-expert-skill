## 한국어 프롬프트 기본

- 현재 플랫폼/화면이 제공한 핸들은 바이트 단위로 보존하고, API 미디어 순번은 검증된 플랫폼 프로필에서 생성합니다. 구조화 역할에는 프롬프트 토큰을 추가하지 않으며 다른 플랫폼의 표기를 추측하지 않습니다.
- 인물 고정, 의상, 공간 구조, 동작의 끝점, 카메라 움직임을 먼저 정합니다.
- “감성적인”, “분위기 있는”, “영화 같은” 같은 단어만 쓰지 말고 카메라, 조명, 재질, 공기, 소리로 분해합니다.
- 최종 자막, 광고 문구, 법적 문구는 영상 생성에 맡기지 않고 편집 단계에서 추가합니다.
- 연속 이야기에서는 이전 승인 클립의 실제 마지막 상태를 기록한 뒤 다음 프롬프트를 작성합니다.

## 연속 클립 템플릿

```text
이야기의 최종 목표: [마지막에 도달해야 하는 상태]
승인된 사실: [이전 영상에서 실제로 일어난 일]
이번 클립만: [이번에 보여줄 하나의 시각적 작업]
아직 보여주지 않을 것: [뒤 클립에 남겨둘 내용]
참조 계획: 인물 이미지는 정체성만, 동영상은 카메라만, 오디오는 템포만 제어하며 타입 바인딩은 현재 플랫폼 프로필로 해석합니다.
프롬프트: [한 가지 동작 + 한 가지 카메라 + 실제 조명 + 소리]
```

## 한국어 예시

타입 바인딩 계획(프롬프트 본문 아님): `product` = 제품 정체성 이미지. 렌더러가 현재 플랫폼에 맞게 바인딩을 해석합니다.

프롬프트 본문:

```text
제품의 로고, 라벨, 병 모양, 색상을 정확히 유지한다.
변화는 작은 물방울이 표면을 따라 내려가는 움직임과 왼쪽에서 지나가는 따뜻한 프랙티컬 조명만 적용한다.
Camera: locked product close-up, slow push-in.
Sound: 낮은 실내 환경음, 마지막에 작은 유리 소리.
자막, 워터마크, 불필요한 글자 추가 금지.
```

## 안전한 재작성

보호된 캐릭터, 실존 인물, 브랜드, 노래, 목소리를 다른 언어로 숨기지 마십시오. 창작 기능은 유지하되 오리지널 인물, 오리지널 세계, 허가된 참조, 또는 후반 작업 대안으로 바꿉니다.
